# Contributing to tern-sdk

Thank you for your interest in contributing to the Synapticode Developer SDK.

## Before You Start

tern-sdk is the public interface to Synapticode's ternary AI stack. The underlying engine, compiler, and runtime are proprietary. Contributions to tern-sdk are welcome in these areas:

- Bug reports and fixes
- Documentation improvements
- Integration examples
- Language binding extensions
- Test coverage

## Reporting Issues

Open an issue on GitHub. Please include:

- SDK version
- Target hardware/NPU
- Minimal reproduction steps
- Expected vs actual behaviour

## Code Contributions

1. Fork the repository
2. Create a feature branch from `main`
3. Write tests for your changes
4. Ensure all tests pass
5. Submit a pull request

## Code Style

- Python: Black formatter, type hints required
- C++: clang-format with project config
- Rust: cargo fmt
- Documentation: Clear, concise, examples preferred
